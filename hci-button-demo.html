AKCB UI Framework

Human-Computer Interaction Compliant Button System for Banking Applications

HCI Principles Implemented

  • Consistency: Unified design patterns across all button states
  • Accessibility: WCAG 2.1 compliant with screen reader support
  • Affordances: Clear visual cues indicating clickable elements
  • Feedback: Immediate visual and haptic response to interactions
  • Error Prevention: Disabled states and loading indicators
  • Touch Targets: Minimum 44px touch targets for mobile devices
  • Cognitive Load: Clear labeling and logical grouping
